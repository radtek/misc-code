This zip file contains the source code examples referenced in Dave Mitchell's article, "Leveraging Your Visual C++ Experience on the Internet with Thin Client Technology" from the December Issue of MSJ. 

A beta version of the framework (ViewSoft Internet) mentioned in the article, as well as additional examples are available for download from http://www.viewsoft.com/examples.

Large portions of the example code were generated by the ViewSoft builder. For information about which lines were written by hand see the examples page. Please note that the ViewSoft Internet beta is required to compile these examples.
